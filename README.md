## PORT PARA ESP32 (SDK-IDF V3.3+)

- FUNC CODE 01 - Read Coils -> OK!
- FUNC CODE 03 - Read Holding Registers -> OK!
- FUNC CODE 06 - Write Single Registers -> OK!
- FUNC CODE 16 - Write Multiple Registers -> OK!
